INTRODUCTION
------------
This module provides a JS widget by updatemybrowser: http://updatemybrowser.org.

If a visitor views your website from an older browser, updatemybrowser will automatically show a banner across the top
of your website to notify the user that there is a new version of their browser available.

* For a full description of the module, visit the project page: https://www.drupal.org/sandbox/i_g_wright/2366113
* To submit bug reports and feature suggestions, or to track changes: https://www.drupal.org/project/issues/2366113

REQUIREMENTS
------------
No special requirements.

INSTALLATION

https://drupal.org/documentation/install/modules-themes/modules-7 for further information.
